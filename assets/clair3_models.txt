Clair3 Model Selection Guide

Model Name                         Platform / Application              Description                                              Notes
---------------------------------------------------------------------------------------------------------------
hifi                               PacBio HiFi                         Standard model for PacBio HiFi sequencing                Recommended for general HiFi data

hifi_revio                         PacBio HiFi Revio                   Model optimised for PacBio Revio platform                Use when data generated on Revio

hifi_sequel2                       PacBio HiFi Sequel II               HiFi model tailored for Sequel II instruments            For Sequel II runs

ilmn                               Illumina short reads                Model for Illumina short-read sequencing                 Experimental / rarely used

ont                                Oxford Nanopore                     Legacy ONT model for older datasets                      Suitable for older basecalling models

ont_guppy5                         Oxford Nanopore (Guppy 5)           Model for reads basecalled with Guppy v5                 Newer ONT datasets

r1041_e82_400bps_sup_v410          ONT R10.4.1 (E8.2 chemistry)        Super-accuracy model for R10.4.1 400bps chemistry        Recommended for modern ONT runs

r1041_e82_400bps_sup_v500          ONT R10.4.1 (E8.2 chemistry)        Updated super-accuracy model (v5.0)                      Not present in some Singularity images

r941_prom_hac_g360+g422            ONT R9.4.1 PromethION HAC           High-accuracy model for R9.4.1 PromethION datasets       HAC basecalling

r941_prom_sup_g5014                ONT R9.4.1 PromethION SUP           Super-accuracy model for PromethION R9.4.1 datasets      Most common SUP configuration


Models can be found in the container by using the following commands:

docker pull quay.io/biocontainers/clair3:1.1.1--py310h779eee5_0
docker run -it --rm quay.io/biocontainers/clair3:1.1.1--py310h779eee5_0 /bin/bash
cd /usr/local/bin/models/
ls
